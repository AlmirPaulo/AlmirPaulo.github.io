Used Technologies:

  • Python

  • XlsxWriter

Mars Rover

This is a page where I work with NASA API to show pictures of Mars taken by the rovers (Spirit, Curiosity and Opportunity). Besides HTML and CSS, here I also work with Brython: a Python lib to work in the front-end.

If you plan to test this website, pay attention to the date you are using. Some of the rovers are inactive, so they do not respond to some "sol" dates.

Used Technologies:

  • HTML
  • CSS
  • Python
  • Brython
  • NASA API

Wiki Scraper

This is a scraper for Wikpedia articles. I made a version for terminal and another one with a GUI (graphic user interface).

Of course I'm also able to save the scraps on files, like an excel file, a ".txt" (as I do on the terminal version of this scraper) or a SQL database. Just don't seems the point once you can download Wikipedia articles in pdf from the website.

Used Technologies:

  • HTML
  • Python
  • Beautiful Soup
  • Pysimple GUI
